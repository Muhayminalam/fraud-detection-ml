# Credit Card Fraud Detection System

End-to-end machine learning application that detects fraudulent credit card transactions using XGBoost, FastAPI, and Streamlit.

## Features

- Trained XGBoost model on 284K+ credit card transactions
- Achieved 0.98 ROC-AUC
- FastAPI REST API for real-time predictions
- Batch prediction endpoint for CSV uploads
- Streamlit dashboard with summary metrics and charts
- Downloadable prediction results

## Tech Stack

Python, Pandas, Scikit-learn, XGBoost, FastAPI, Streamlit, Joblib
